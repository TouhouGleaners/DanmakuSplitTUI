# DanmakuSplitCLI

基于时间轴排序的 B 站弹幕 XML 分割工具。

### 1. 核心命令 (Quick Start)

**场景 A：按人头平分 (领区制)**
```bash
python main.py -i input.xml -u "张三 李四"
```

**场景 B：按数量分割 (定量制)**
```bash
python main.py -i input.xml -l 3000
```

**场景 C：混合分配 (指定 P 号)**
```bash
python main.py -i input.xml -u "张三 李四" -l 3000 -p 2
```

### 2. 参数说明
| 参数 | 说明 | 例子 |
| :--- | :--- | :--- |
| `-i` | 输入 XML 路径 (必填) | `-i "P1.xml"` |
| `-u` | 负责人名单 (空格分隔) | `-u "张三 李四"` |
| `-l` | 单个文件弹幕上限 | `-l 500` |
| `-p` | 视频分 P 号 (默认 1) | `-p 3` |

### 3. 标准协作流程 (SOP)
1.  **切割**：执行命令，程序自动在 XML 同级目录生成 `Split_...` 文件夹。
2.  **存证**：截图终端打印的**任务分配表**并留存。
3.  **分发**：根据文件名后缀 `_[负责人].xml` 进行分发。
4.  **校验**：统筹人核对表中**时间区间**的**起始锚点**弹幕内容是否一致。

### 4. 文件名逻辑
- **分配给人时**：`P1_Part1_[负责人].xml`
- **纯按数切时**：`P1_Part1.xml`
